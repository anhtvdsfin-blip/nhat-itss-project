# nhat-itss-project

**Nhóm 10** - JT-Helper

## Giới thiệu
Ứng dụng hỗ trợ học tiếng Nhật với các tính năng:
- Ghi âm và chuyển đổi giọng nói thành văn bản
- Phân tích nội dung tiếng Nhật
- Dịch thuật
- Giải thích từ vựng

## Cấu trúc dự án
- `backend/` - API backend (Node.js + Express) - Deploy trên Vercel
- `react-app/` - Frontend (React + Vite) - Deploy trên Vercel

## Backend URL
- Production: https://nhat-itss-project-be.vercel.app/api

## Cài đặt và chạy

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd react-app
npm install
npm run dev
```
